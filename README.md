# Google recaptcha v3

- Run
Go to folder code and start server
```
$ php -S 127.0.0.1:8080
```

- Paste **GOOGLE_RECAPTCHA_V3_SITE_KEY** and **GOOGLE_RECAPTCHA_V3_SECRET_KEY** to code

- In browser, test
```
http://localhost:8081/
```

- You can check `$resultJson` in `action.php` to view recaptcha result
